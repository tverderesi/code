JavaScript em Arquivos Externos

Os scripts JavaScript também podem ser carregados em arquivos externos. Isso facilita a leitura do código HTML, uma vez que os scripts ficam escritos em um arquivo separado e carregá-lo utilizando a tag <script> e, além disso, os scripts podem ficar carregados em cache ao se fazer o recarregamento da página. Observe que o botão continua funcionando.
