Aula 05

Var, Let, Const

  • var, let e const são variáveis, como matemática;
  • precisamos declarar variáveis, por exemplo: var x = 3, declara uma variável chamada x com 3;
  • var é uma declaração antiga, sendo utilizada até 2015; let é utilizada como sua subsituição, o exemplo se torna let x = 3;
  • Algumas palavras são reservas à linguagem, como let, var, if, function, etc...;
  • Podemos verificar esses valores utilizando o comando console.log();
  • A criação de variáveis segue algumas regras, como:
    • Não se pode usar palavras reservadas;
    • Não se podem começar por números;
    • Não podem se usar espaços;
    • nomes de variáveis são case-sensitive;
    • Não se pode declarar uma variável mais de uma vez, se isso acontecer, a variável será sobrescrita;
    • Não se pode utilizar caracteres especiais;
    • Não é comum utilizar acentuação nas variáveis;
  • Para valores que não mudam, utilizamos a declaração const. Isto é útil quando queremos evitar que essa variável seja sobrescrita. Um bom exemplo é const pi = 3.14;
